Site Plan: Smart Building Systems Explorer

1. Site Name

Smart Building Systems Explorer

Reason: This name clearly communicates the purpose of the site—to allow users to explore commercial buildings and the automation systems they use. “Explorer” signals browsing and discovery, while “Smart Building Systems” reflects the technical focus on HVAC, controls, and energy monitoring.

2. Site Purpose

This website will:

  • Educate visitors about modern commercial building automation systems.
  • Showcase example buildings and the technologies they use (HVAC platforms, controllers, monitoring).
  • Allow users to filter and/or sort buildings by system type and features.
  • Provide a form for requesting a building automation assessment.
  • Demonstrate responsive design, accessibility, and data-driven content using HTML, CSS, and JavaScript.

3. Scenarios

Scenario 1

A facilities manager wants to compare different HVAC automation platforms used in office buildings so they can understand upgrade options before contacting an integrator.

Scenario 2

A building automation student is researching smart building technologies and wants to browse example buildings and view details about controllers, sensors, and energy monitoring systems.

4. Color Schema

The following colors will be used across the site. This site plan document also uses these colors (headings, accents, and layout backgrounds) to satisfy the rubric requirement.

Color Hex Where Used
Deep Blue #1a73e8 Primary buttons, links, headings accents, interactive states
Slate Gray #2f3e46 Header/footer backgrounds, section titles, key UI regions
Light Gray #f4f6f8 Page background, subtle section separation
White #ffffff Cards, content panels, readable surfaces
Accent Green #3cb371 Highlights, success messages, small emphasis elements

Color Application Notes

  • Buttons and primary links will use Deep Blue with white text.
  • Site chrome (header/footer) will use Slate Gray for strong contrast.
  • Cards sit on Light Gray background for clean separation.
  • Accent Green used sparingly for “success/complete” indicators.

5. Typography

Primary Font: Noto Sans (Google Fonts)

  • Headings (h1–h3): Bold weights for hierarchy and scanning.
  • Body text: Regular weight for readability.
  • Navigation: Semi-bold for clarity.
  • Buttons: Semi-bold for emphasis.

This site plan is displayed using Noto Sans to demonstrate typography usage in the document.
